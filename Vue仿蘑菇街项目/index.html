Vue仿蘑菇街项目

发表于 2021-11-30 更新于 2021-12-05

​ 这是根据王元红老师指导完成的类似于蘑菇街的电商APP。后端数据接口由王元红老师提供，课程指路最全最新Vue、Vuejs教程，从入门到精通

1、项目描述

  • 项目基于VueCLI4脚手架 + Vue全家桶 + ES6 + Webpack等技术开发

    • vue-cli 安装：npm install -g @vue/cli

      vue create 项目名

    • vueRouter 安装：npm install vue-router

    • vuex 是专门为vue.js应用程序开发的状态管理模式

      管理共享状态 大型单页面应用

      安装 ： npm install vuex –save

    • axios是一个基于promise的http库，可以用在浏览器和node.js中

  • 项目采用模块化、组件化、工程化的模式开发

  • 项目主要分为四大模块：首页 + 分类 + 购物车 + 我的

2、使用的插件或第三方库

  • 使用 vue-router 开发单页面

  • 使用 vuex 管理应用组件的状态

  • 使用 axios 的封装从后端拿取数据

  • 使用 better-scroll 实现移动端页面滑动效果

  • 使用mitt实现$bus事件总线

3、实现的功能

  • 首页商品款式的切换
  • 首页到详情页的跳转
  • 右下角回到顶部
